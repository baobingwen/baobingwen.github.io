二是类似于html语言，使用尖括号br尖括号

加粗

加粗的方式为在要加粗的部分两边各加两个星号或下划线，但是不推荐使用下划线，只使用星号最好
例如：__这是粗体__，这是一部分粗体

斜体

斜体的方式为在要变斜的部分两边各加一个星号或下划线，但是不推荐使用下划线，只使用星号最好
例如：这是斜体，这是一部分斜体

粗体和斜体

    代码语法

    对于单词或短语，包裹在反引号 () 中 例如：At the command prompt, type nano`.

    转义反引号

    代码中包含一个或多个反引号，将其包裹在双反引号()中 例如：Use code in your Markdown file.``

    代码块

    将代码块的每一行缩进至少 四个空格 或 一个制表符

    分割线语法

    单独一行上 使用 三个或多个 星号 、破折号或下划线 ，不能包含其他内容。
    例如：
    ****** --- ___

    注：为了兼容性，请在分隔线的前后均添加空白行。

    链接语法

    超链接Markdown语法代码：[超链接显示名](超链接地址 "超链接title")

    例如：
    这是洛谷链接 洛谷。
    这是洛谷链接 洛谷。

    给链接加Title

    链接title是当鼠标悬停在链接上时会出现的文字，这个title是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。
    例如：
    这是一个链接 西电oj。
    这是一个链接 西电oj。

    网址和Email地址

    使用尖括号可以很方便地把URL或者email地址变成可点击的链接。
    例如：
    https://www.luogu.com.cn
    fake@example.com
    https://www.luogu.com.cn
    fake@example.com

    带格式化的链接

    强调或斜化链接在链接语法前后增加星号
    例如：

    ``**我爱[luogu](https://www.luogu.com.cn)。**``

    我爱luogu。
    要将链接表示为代码，在方括号中添加反引号‘`’。
    例如：

    我爱['luogu'](https://www.luogu.com.cn)。

    我爱‘luogu’。
